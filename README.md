## Arabic

### 🌟 ما هو فِ البايو؟

فِ البايو هو تطبيق ويب مجاني ومفتوح المصدر يتيح لك إنشاء صفحة شخصية تحتوي على جميع روابطك المهمة في مكان واحد. تم تصميمه خصيصاً للمحتوى العربي مع دعم كامل للغة العربية واتجاه الكتابة من اليمين إلى اليسار.

### ✨ المميزات

- 🌐 **دعم اللغة العربية**: تصميم مُحسّن للغة العربية مع دعم RTL
- 🔄 **تبديل اللغات**: إمكانية التبديل بين العربية والإنجليزية
- 🎨 **مظاهر متعددة**: اختر من بين المظهر الافتراضي أو الداكن أو الملون
- 📱 **تصميم متجاوب**: يعمل بشكل مثالي على جميع الأجهزة
- 💾 **حفظ محلي**: يحفظ بياناتك في متصفحك
- ⚡ **سريع وخفيف**: لا يحتاج لخادم، يعمل كملف HTML بسيط
- 🔧 **قابل للتخصيص**: غيّر الاسم والصورة والنبذة والروابط

### 🚀 كيفية الاستخدام

1. **افتح الملف**: افتح `index.html` في أي متصفح
2. **خصص صفحتك**: استخدم قسم "تخصيص الصفحة" لإضافة:
   - اسمك
   - نبذة عنك
   - صورتك الشخصية
   - روابطك المفضلة
3. **اختر المظهر**: جرب المظاهر المختلفة
4. **احفظ**: كل التغييرات تُحفظ تلقائياً في متصفحك

### 🛠️ التثبيت والاستضافة

#### للاستخدام المحلي:
```bash
git clone https://github.com/aldoyh/fel-baio.git
cd fel-baio
# افتح index.html في متصفحك
```

#### للاستضافة على GitHub Pages:
1. Fork هذا المشروع
2. اذهب إلى Settings → Pages
3. اختر المصدر: Deploy from a branch
4. اختر الفرع: main
5. صفحتك ستكون متاحة على: `https://username.github.io/fel-baio`

#### للاستضافة على Netlify:
1. اسحب مجلد المشروع إلى Netlify
2. أو اربط مع GitHub repository
3. صفحتك ستكون جاهزة في ثوانٍ

## English

### 🌟 What is Fel-Baio?

Fel-Baio is a free and open-source web application that allows you to create a personal page containing all your important links in one place. It's specifically designed for Arabic content with full Arabic language support and right-to-left (RTL) direction.

### ✨ Features

- 🌐 **Arabic Language Support**: Optimized design for Arabic with RTL support
- 🔄 **Language Switching**: Switch between Arabic and English
- 🎨 **Multiple Themes**: Choose from default, dark, or colorful themes
- 📱 **Responsive Design**: Works perfectly on all devices
- 💾 **Local Storage**: Saves your data in your browser
- ⚡ **Fast & Lightweight**: No server needed, works as a simple HTML file
- 🔧 **Customizable**: Change name, image, bio, and links

### 🚀 How to Use

1. **Open the file**: Open `index.html` in any browser
2. **Customize your page**: Use the "Customize Page" section to add:
   - Your name
   - Your bio
   - Your profile picture
   - Your favorite links
3. **Choose theme**: Try different themes
4. **Save**: All changes are automatically saved in your browser

### 🛠️ Installation and Hosting

#### For local use:
```bash
git clone https://github.com/aldoyh/fel-baio.git
cd fel-baio
# Open index.html in your browser
```

#### For GitHub Pages hosting:
1. Fork this project
2. Go to Settings → Pages
3. Select source: Deploy from a branch
4. Select branch: main
5. Your page will be available at: `https://username.github.io/fel-baio`

#### For Netlify hosting:
1. Drag the project folder to Netlify
2. Or connect with GitHub repository
3. Your page will be ready in seconds

### 🎯 Use Cases

- **Influencers**: Gather all your links in one place
- **Small Businesses**: Simple page to showcase your services
- **Developers**: Display your projects and accounts
- **Artists**: Link between your work and platforms
- **Education**: Share resources and useful links
